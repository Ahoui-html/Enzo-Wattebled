Made during a one week workshop, this interactive installation run strange attractor functions to generate visuals.

The graphic part is linked to an autogenerated audio, using chaotic values of the function. The result is a deep and chaotic visual and audio landscape reacting the user input.

“ The theory of chaos“ by James Gleick was our main source of inspiration in this project.
